# ArenaSync

Application React pour la gestion et l'affichage de tournois sportifs.

## Composants

Header → Barre supérieure avec recherche  
StatusBadge → Badge dynamique de statut  
ParticipantRow → Ligne d'un participant  
TournamentCard → Carte détaillée d'un tournoi  

## Fonctionnalités

- Affichage dynamique des tournois
- Filtrage par sport
- Tabs Info / Participants
- Liste dynamique des participants
- Interface responsive
